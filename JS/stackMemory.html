    # 堆栈内存

    # 例子1

    let a = {n: 1};
let b = a;
a.x = a = {n: 2};
console.log(a.x);  // undefined
console.log(b);  // {n: 1, x: {n: 2}}

    # 计算注意点

    • 所有的赋值操作，先创建值，再创建变量
    • 所有的引用类型存储不是存储在栈中，是存储在堆内存中的
    • 所有带突出成员访问的，都要优先处理

    # 例子2

    var a = 0;
if (true) {
    a = 1;
    function a() {};
    a = 21;
    console.log(a)
}
console.log(a);

    # 计算注意点

    变量提升：当前上下文代码执行之前，会把var/function声明加定义，带var的只声明，带function的是声明+定义 如果遇到{}新老浏览器表现不一致

    # 【IE浏览器 <= IE10】

    不管{}，还是一如既往的function声明+定义，而且也不会存在块级作用域

    # 【新版本浏览器】（兼容ES3、兼容ES6）

    {}中的function，在全局下只声明不再定义 {}中出现function/let/const会创建一个块级上下文

    # 例子3

    var x = 1;
function func(x, y = function anonymous1() {x = 2}) {
    x = 3;
    y();
    console.log(x);
}
func(5);
console.log(x);
